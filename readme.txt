== Installation ==

1. Visit 'Plugins > Add New'.
1. Search for 'Gutena Accordion' and install it.
1. Or you can upload the `gutena-accordion` folder to the `/wp-content/plugins/` directory manually.
1. Activate Gutena Accordion from your Plugins page.
